## 🌟 Highlights & Features

- **Bilingual UI**: Seamlessly switch between English and Bangla (বাংলা).
- **Catppuccin Theming**: Beautiful token-based theming with Light/Dark mode support.
- **AI Chatbot**: Floating AI chatbot powered by Vercel AI SDK and OpenRouter with streaming responses.
- **Live Stats**: Dynamic floating panel showcasing live competitive programming stats (Codeforces + LeetCode).
- **Responsive & Animated**: Mobile-first design with smooth Framer Motion animations and Lenis scroll.
- **Contact API**: Fully functional contact form integrated with Resend.

## 🛠️ Getting Started

### Prerequisites

- [Bun](https://bun.sh/) `>= 1.3`
- Node.js `>= 20` (recommended for toolchain compatibility)

### Installation

```bash
git clone https://github.com/SnoOpWo0t/portfolio-dotg.git
cd portfolio-dotg
bun install
cp .env.local.example .env.local
```

### Environment Variables

Set these in `.env.local`:

```env
OPENROUTER_API_KEY=your_openrouter_api_key
RESEND_API_KEY=your_resend_api_key

# Optional (Defaults provided in code)
RESEND_FROM=Afrin Portfolio <onboarding@resend.dev>
RESEND_TO=afrinsultanaakhi138@gmail.com
GITHUB_TOKEN=your_github_token
```

### Run Locally

```bash
bun run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
